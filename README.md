# 365生活服务平台


## 项目简介

本项目是365家政的后台管理系统。为客户端项目提供后台支持，同时满足公司日常的业务需求。[项目入口](https://manage.sy365.cn)
## 权限管理
权限模块的设计基于RBAC（Role-Based Access Control，基于角色的访问控制）
